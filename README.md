# 🚗 Logical Car Parking System

A hardware-based smart parking system built using logic gates, IR sensors, and LED indicators — designed to detect parking space availability and guide drivers in real time with minimal hardware.


---

## 📌 Project Overview

This project implements a smart car parking system entirely using digital logic components. IR sensors detect whether a parking slot is occupied or vacant, logic gates process the sensor signals, and LEDs provide visual feedback to drivers, indicating available and occupied spaces at a glance.

The system improves parking efficiency, reduces congestion, and demonstrates how real-world automation problems can be solved with fundamental digital electronics.

---

## ⚙️ Components Used

- IR Sensors — detect vehicle presence in each parking slot
- Logic Gates — process sensor signals to determine slot status
- LED Indicators — display occupied (red) / vacant (green) status
- Additional electronic components (resistors, wires, power supply, breadboard/PCB)

---

## 🏗️ How It Works

1. Each parking slot has an IR sensor that detects the presence of a vehicle.
2. Sensor outputs are fed into a logic gate circuit that processes the occupancy state.
3. LEDs corresponding to each slot light up to indicate whether the slot is **occupied** or **vacant**.
4. The system provides real-time visual guidance to drivers entering the parking area.
